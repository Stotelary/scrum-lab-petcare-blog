## Reflexión
### - ¿Cómo se sintieron al trabajar con roles definidos?
Es un poco extraño al ser primera vez, pero bastante cómodo porque te permite conocer los límites de los demás, llevar un registro de lo que se ha ido avanzando y facilitar la comunicación.
### - ¿Qué aprendieron sobre colaboración y comunicación?
Es muy importante ya que facilita el flujo de trabajo y la resolución de problemas inesperados. Una reunión inicial es vital para saber cómo funciona el equipo, las habilidades de cada uno y otros aspectos importantes.
### - ¿Qué cambiarían para su siguiente Sprint?
Probar nuevos roles y distribuciones de tareas para experimentar y descubrir cuál calza mejor con cada integrante.
